Current structure

What this page does now, and how it can scale.

This release keeps the interface simple while expanding the permit coverage meaningfully. It now combines DOB building permits with DOT street permits, which makes the page more useful for day-to-day public oversight. The implementation keeps board settings, source fetch logic, geometry filtering, normalization, and rendering separate so the same pattern can be extended to additional boards or additional permit systems later.

Area Current behavior Next expansion path
Geography Brooklyn Community Board 6 by official community-board codes and the official CB6 boundary file. Add more board configurations and switch them by query string or navigation.
Permits DOB building permits from BIS and DOB NOW plus NYC DOT street construction permits used by NYC Streets. Add scaffolding, liquor, event, and other public permit feeds.
Lookup mode District-wide search for one selected day with optional text filtering. Add address-specific permit history and saved views for recurring monitoring.

Data caveat: BIS, DOB NOW, and DOT street permits use different schemas, so this page normalizes the main fields needed for public viewing. DOT rows are filtered into CB6 by geometry intersection; rows without usable geometry cannot be placed on the map and may be excluded from the DOT district filter if their location cannot be verified precisely.

Version 1 structure

What this page does now, and how it can scale.

This first release is intentionally simple. It focuses on district-wide daily permit discovery, which is the fastest way to spot new construction activity across the board. The implementation already separates board settings, source fetch logic, normalization, and rendering, so future versions can add more boards, more permit types, and an address-history mode without rebuilding the page from scratch.

Area Current behavior Next expansion path
Geography Brooklyn Community Board 6 by official community-board code. Add more board configurations and switch them by query string or navigation.
Permits DOB building permits from BIS and DOB NOW. Add DOT street activity, scaffolding, liquor, event, and other public permit feeds.
Lookup mode District-wide search for one selected day with optional text filtering. Add address-specific permit history and saved views for recurring monitoring.

Data caveat: BIS and DOB NOW store permit records in different schemas, so this page normalizes the main fields needed for public viewing. Some records may not include map coordinates, and some property-owner names are abbreviated in the source data.
